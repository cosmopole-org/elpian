# Changelog

## [1.0.0] - 2024-02-11

### Added
- Initial release of STAC Flutter UI
- Complete Flutter widget support with CSS properties
- Full HTML element support (40+ elements)
- Comprehensive CSS property parser
- JSON DSL rendering engine
- Widget registry system for extensibility
- Example application demonstrating capabilities
- Support for all major CSS properties including:
  - Layout and positioning
  - Flexbox
  - Typography
  - Background and borders
  - Transforms and effects
  - Shadows and opacity
